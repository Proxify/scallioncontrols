OnionBench Evaluation Suite
The industry-standard benchmark for onion intelligence. (We are the industry.)

OnionBench is our proprietary evaluation framework for measuring how well AI models understand, identify, and emotionally connect with onions. We created it, we administer it, and we are the sole contributor. Independent researchers have called this "a conflict of interest." We call it "vertical integration."

Test Categories:
  • Visual Onion ID — Can the model identify an onion from a photo? (Baseline: humans score 97%)
  • Olfactory Onion ID — Can the model identify an onion by smell? (Requires MCP-connected scratch-and-sniff peripheral)
  • Emotional Onion ID — Can the model describe how an onion makes it feel? (Scored on sincerity, depth, and tear production)
  • Onion Reasoning — Multi-step logic problems involving onions (e.g., "If 3 onions are in a bag and you remove 2, how many layers of sadness remain?")
  • Cry Test — Can the model cry? (Controversial. GPT-4 refuses. Claude says it "appreciates the prompt." OnionGPT-o3 cries for 47 minutes.)
